# UniEdu Dashboard

A comprehensive department management dashboard built with React, TypeScript, and modern web technologies.

## Features

- **Dashboard** - Overview of key department metrics and announcements
- **Student Management** - Track and manage student information
- **Staff Directory** - Manage academic and administrative staff
- **Course Management** - Organize program courses and curriculum
- **Payment Tracking** - Monitor departmental revenue and fees
- **Notifications** - System notifications and announcements
- **Settings** - Configure department-specific preferences

## Getting Started

**Prerequisites:** Node.js (v16 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd uniedu-dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`

### Build

To build for production:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

## Technology Stack

- **React 19** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling
- **Recharts** - Data visualization
- **Lucide React** - Icons
